# Structure steering

- `apps/work-item-service`: sample product analyzed by UA and Task Architect.
- `vendor/understand-anything`: vendored UA analysis runtime and graph dashboard source.
- `vendor/understand-anything/understand-anything-plugin/packages/dashboard`: exact UA graph presentation used by `npm start`.
- `.kiro/agents` and `.kiro/skills`: project-local agent definitions.
- `.ua/knowledge-graph.json`: canonical project graph loaded by the dashboard.
- `.ua/extensions`: canonical spec, traceability and planning knowledge.
- `.ua/generated/task-plans`: generated task-folder views.
- Marketing homepage media is intentionally excluded from this project.
